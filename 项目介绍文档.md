## 一、项目详情及使用方式

### 项目概述

PartJava 智能学习平台是一个面向计算机学习者的综合性在线学习平台，集成了 AI 智能助手、3D 可视化、在线代码执行、智能笔记管理等核心功能。平台覆盖 15 个技术领域，旨在通过现代化 Web 技术和人工智能，为学习者提供系统化、沉浸式的编程学习体验。

- **在线地址**：[https://partjava.com](https://partjava.com)
- **GitHub**：[https://github.com/partjava/i](https://github.com/partjava/i)
- **当前版本**：v1.3.0
- **项目状态**：✅ 生产环境运行中

## 二、项目亮点

### 亮点一：原创 AI 3D 机器人（史迪奇风格）

平台自主开发了一个基于 Three.js + WebGL 的 3D 虚拟角色，采用史迪奇卡通风格设计，拥有流畅的动画效果和交互式控制面板。这不是调用任何第三方模型资产，而是完全通过代码构建的 3D 角色系统，在国内同类学习平台中极为罕见。

### 亮点二：原创 EyePet 互动虚拟宠物

平台独立开发了一个纯 SVG + CSS 实现的虚拟宠物角色，具备以下交互能力：
- 眼睛实时追踪鼠标位置，瞳孔跟随移动，带高光细节

- 随机弹跳动画 + 持续悬浮漂浮效果
- 腮红、身体高光、地面阴影等精细视觉细节

这是完全原创的前端交互设计，无任何第三方动画库依赖。

## 三、AI 技术落地使用情况

### 总体概述

本项目将 AI 技术真实落地于多个核心功能模块，全部部署在生产环境中稳定运行，并非停留在概念或演示层面。以下逐一介绍各模块的 AI 技术应用情况。

### 1. DeepSeek API — 智能问答与多轮对话

平台后端集成 DeepSeek 大语言模型 API，通过 Python FastAPI 服务封装调用逻辑，以 Docker 容器化独立部署，与前端 Next.js 服务解耦，通过 Nginx 反向代理统一对外暴露 `/api/ai/` 接口。

用户可在 AI 聊天页面（`/ai-chat`）与智能助手进行多轮对话，支持提问编程问题、寻求代码解释、获取学习路径建议等场景。每次对话的历史消息通过数据库持久化存储，用户可随时查看历史会话记录，不会因刷新页面而丢失上下文。平台还提供 AI 辅助功能页面（`/ai-assistant`），集成了更多 AI 辅助学习的入口。

![AI多轮对话界面](/images/images/ai-conversation.png)
*截图：AI 多轮对话界面，展示历史会话记录*

### 2. Judge0 CE API — 云端代码执行沙箱

代码执行功能通过接入 Judge0 CE 云端代码评测引擎实现，是平台"边学边练"能力的核心支撑。用户在代码编辑器中编写代码并点击运行后，后端将代码内容、语言类型、标准输入数据打包，通过 RapidAPI 发送至 Judge0 CE 沙箱环境执行。

后端采用异步轮询机制获取执行结果（最多轮询10次，每次间隔1秒），并对返回的 Base64 编码输出进行自动解码处理，最终将执行结果、错误信息、运行时间、内存占用一并返回给前端展示。整套机制实现了安全隔离的代码运行能力，目前支持 Python、JavaScript、Java、C++、C、Go、PHP、Rust、C#、TypeScript 共 10 种编程语言，并支持标准输入（stdin）。


### 3. AI/ML 算法 3D 可视化教学

算法可视化系统内置了完整的 AI 与机器学习算法的 3D 动态演示模块，基于 Three.js 构建，帮助学习者直观理解抽象的 AI 算法原理：

- **K-Means 聚类**：动态展示数据点随迭代轮次逐步分组的过程，质心位置实时更新移动
- **线性回归**：在 3D 空间中实时绘制数据点散布与回归线的拟合过程
- **神经网络**：以 3D 层级结构展示网络节点、连接权重与信号前向传播路径
- **梯度下降**：可视化损失函数曲面，动态展示参数沿梯度方向的更新路径
- **决策树**：动态构建树形分类结构，逐步展示节点分裂与分类决策过程

这些可视化内容是平台 AI 学习模块的重要组成部分，将原本只能靠文字和公式理解的算法，转化为可交互的 3D 动画演示。


### 4. 智能成就系统 — 数据驱动的行为分析

成就系统通过后端 API 持续分析用户的学习行为数据，自动判断成就解锁条件并更新进度，无需用户手动触发。系统追踪的行为维度包括：笔记发布数量、连续学习天数、每日学习开始时间段（用于判断早起鸟儿/夜猫子）、公开分享次数、笔记浏览量、获赞数量、覆盖的技术领域数量等。15 个成就形成完整的激励体系，通过数据驱动的方式引导用户持续学习、深度参与。

### 5. 实时学习行为追踪与热力图

平台以 60 秒为间隔自动向后端同步当前用户的学习时间数据（`/api/study/sync-time`），结合学习记录接口（`/api/study/record`）形成完整的学习行为时间线。这些数据最终汇聚到学习热力图中，以类 GitHub 贡献图的形式可视化展示用户每天的学习活跃度，让学习坚持情况一目了然。学习统计数据同时在个人中心、数据大屏等多处展示，为后续个性化推荐功能的实现奠定了完整的数据基础。

### 6. 平台学习内容体系 — 15个技术领域全覆盖

平台主页构建了完整的计算机学习内容体系，覆盖 15 个核心技术领域，每个领域均有独立的学习页面和详细的知识框架：

**编程语言**（Java、Python、C++、Go、PHP）：涵盖 Java 面向对象与企业级开发、Python 数据分析与机器学习、C++ 系统编程与算法、Go 并发编程与微服务、PHP Web 后端开发，每种语言从基础语法到高级特性系统讲解。

**前端技术**：HTML5 语义化标签、CSS3 响应式设计与动画、React 组件化开发与状态管理、Vue.js 渐进式框架生态，配合实战项目（个人作品集、电商平台前端、管理后台）。

**后端开发**：Node.js 全栈开发、Spring Boot Java 企业级框架、Django Python Web 框架、Express 轻量级框架，重点讲解 RESTful API 设计、微服务架构、性能优化与安全防护。

**数据库**：MySQL 关系型数据库与事务处理、MongoDB 文档数据库、Redis 缓存数据库、SQL 查询优化与索引设计，配合电商系统数据库设计等实战案例。

**人工智能**：机器学习算法原理与模型训练、深度学习神经网络与 TensorFlow、自然语言处理与语言模型、计算机视觉与 OpenCV，配合智能推荐系统、图像分类器、聊天机器人等项目实战。

**网络安全**：密码学加密算法与数字签名、渗透测试漏洞发现与安全评估、防火墙与入侵检测，包含 CTF 竞赛题目和安全漏洞分析实践内容。

**软件工程**：系统架构设计与模块化、单元测试与集成测试、DevOps 持续集成与自动化部署、敏捷开发与团队协作。

**操作系统**：Linux 命令行与系统管理、Windows 系统配置与服务管理、进程管理与线程同步、虚拟内存与垃圾回收机制。

**网络技术**：TCP/IP 网络分层与协议栈、HTTP/HTTPS Web 协议与安全传输、WebSocket 实时通信、CDN 内容分发与性能优化。

**物联网**：传感器技术与数据采集、MQTT/CoAP/LoRa 通信协议、边缘计算与实时响应，配合物联网应用开发实战项目。

**数据结构**：经典数据结构原理与实现，配合算法可视化系统直观演示。

**计算机组成原理**：计算机底层原理与硬件架构知识体系。

每个领域均配备项目实战页面，提供 Go 语言项目实战、SQL 数据库实战、Java 企业级开发、Python 数据分析、C++ 系统编程、操作系统实践、物联网应用开发等多个可实际操作的项目案例，真正做到学以致用。

## 四、项目涉及的关键技术

### 前端核心技术

**Next.js 14 App Router**
采用 Next.js 14 的 App Router 架构，利用 React Server Components 和客户端组件混合渲染，实现页面快速加载和 SEO 优化。动态路由支持学习内容、用户主页、挑战详情等多种页面结构。

**Three.js + WebGL 3D 渲染**
3D 机器人和算法可视化系统均基于 Three.js 构建。通过 WebGL 在浏览器中实现硬件加速的 3D 渲染，包括自定义几何体、材质、光照、动画系统，以及复杂数学曲面（Mandelbrot 集、Klein 瓶、Lorenz 吸引子等）的实时计算与渲染。

**Monaco Editor**
集成 VS Code 同款编辑器内核 Monaco Editor，提供专业级代码编辑体验，支持语法高亮、智能提示、多语言支持，与 Judge0 CE 执行引擎配合实现完整的在线 IDE 功能。

**ECharts 数据可视化**
使用 ECharts 构建数据大屏和学习热力图，支持实时数据更新和响应式图表布局，多维度展示平台统计数据和用户学习行为数据。

**Tailwind CSS + Ant Design**
双 UI 体系并行：Tailwind CSS 负责自定义样式和响应式布局，Ant Design 提供表单、表格、模态框等复杂组件，两者协同覆盖全部 UI 需求。

**PWA（渐进式 Web 应用）**
通过 Service Worker 和 Web App Manifest 实现 PWA 支持，用户可将平台安装到桌面，支持离线页面展示，提升移动端使用体验。

### 后端核心技术

**NextAuth.js 认证系统**
基于 NextAuth.js 实现完整的认证体系，包括 JWT 令牌管理（30天有效期）、会话状态管理、密码加密存储（bcrypt）、密码重置 token 机制（crypto 模块生成，24小时有效）。

**MySQL + 连接池**
生产数据库采用 MySQL 8.0，设计了用户表、笔记表、学习数据表、评论表、学习时间表等核心表结构，通过连接池管理数据库连接，保障高并发下的稳定性。

**Redis 缓存**
引入 Redis 作为缓存层，对高频查询数据进行缓存，降低数据库压力，提升接口响应速度。

**Python FastAPI — AI 服务**
AI 服务独立部署为 Python FastAPI 应用，封装 DeepSeek API 调用逻辑，通过 Docker 容器化运行，与 Next.js 主服务解耦，通过 Nginx 反向代理统一对外暴露接口。

**Judge0 CE API — 代码执行引擎**
通过 RapidAPI 接入 Judge0 CE，实现安全隔离的代码执行沙箱。后端处理语言 ID 映射、Base64 编解码、异步轮询执行状态等逻辑，支持 10 种编程语言。

### 部署与运维技术

**Nginx 反向代理**
Nginx 作为统一入口，将前端请求代理至 Next.js（端口 3000），将 AI 接口请求代理至 FastAPI（端口 8000），并配置 SSL 证书实现 HTTPS 访问。

**PM2 进程管理**
Next.js 生产服务通过 PM2 管理，支持进程守护、自动重启、日志管理，保障服务稳定运行。

**Docker 容器化**
AI 服务以 Docker 容器运行，实现环境隔离和一键部署，避免依赖冲突。

**Let's Encrypt SSL**
通过 Let's Encrypt 申请免费 SSL 证书，配置 HTTPS 加密访问，证书自动续期，保障数据传输安全。

### 前端工程化技术

**TypeScript 全量类型覆盖**
前端代码全量使用 TypeScript，定义了完整的接口类型、组件 Props 类型、API 响应类型，有效减少运行时错误，提升代码可维护性。

**Repository 模式 + Service 层**
后端 API 采用分层架构：Repository 层负责数据库操作（BaseRepository、NoteRepository、UserRepository），Service 层封装业务逻辑（NoteService、UserService、CommentService、StatsService），API 路由层只负责请求处理，职责清晰。

**请求缓存与防抖优化**
搜索功能实现防抖优化，避免频繁请求；API 层引入请求缓存机制（requestCache），减少重复数据库查询。

**Web Vibration API**
移动端手势操作触发 Web Vibration API，提供触觉反馈，增强移动端交互体验。

## 七、项目原创性说明

本项目为**纯原创作品**，并非在其他项目基础上优化或二次开发，所有功能模块均从零设计和实现。

以下是各核心模块的原创说明：

**AI 3D 机器人**
未使用任何现成的 3D 模型资产或角色库，完全通过 Three.js 代码构建几何体、材质、骨骼动画系统，史迪奇风格的外观设计也是原创。

**EyePet 虚拟宠物**
未使用任何动画库或 Canvas 框架，纯 SVG 矢量图形 + CSS 动画 + React Hooks 实现，眼睛追踪算法、心情状态机、弹跳动画均为原创逻辑。

**算法可视化系统**
20+ 种算法的 3D 可视化渲染逻辑全部原创实现，包括每种算法的步骤分解、Three.js 场景构建、动画帧控制，以及 Mandelbrot 集、Klein 瓶等复杂数学曲面的实时计算渲染。

**平台整体架构**
从数据库表结构设计、Repository/Service 分层架构、NextAuth.js 认证集成、Judge0 CE 代码执行封装，到 Nginx + PM2 + Docker 的生产部署方案，均为团队独立设计和实现。

**UI 与交互设计**
翻转卡片工具导航、移动端手势系统、成就徽章体系、学习热力图等交互设计均为原创，未参考或复制任何现有平台的设计方案。
